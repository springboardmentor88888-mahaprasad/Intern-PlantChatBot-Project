# Intern Plant ChatBot Project

This repository contains the complete project source code for the Plant Disease Detection ChatBot using Streamlit and Deep Learning.

## Mentor
Mahaprasad Jena

## Intern Guidelines
- Each intern must create their own branch.
- All work must be committed under personal branches.
- No code should be pushed directly to main.
- Use GitHub for documentation & file sharing.

## How to Run
pip install -r requirements.txt
streamlit run app.py
